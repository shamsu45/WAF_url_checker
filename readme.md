🔐 Web Application Firewall (WAF) - URL Scanner

This project is a simple Web Application Firewall built using Python Flask. It scans user-submitted URLs for potentially malicious patterns using regular expressions and provides real-time feedback. It also includes admin features to view and reset blocked URLs and logs.

---

## 🚀 Features

- ✅ Detects suspicious/malicious URLs using regex pattern matching
- ✅ Real-time result display: Safe or Malicious
- ✅ Logs all scanned URLs (`waf_log.txt`)
- ✅ Admin panel to:
  - View blocked URLs
  - View log file contents
  - Reset logs and clear blocked URLs

---

## 🛠️ Technologies Used

- **Python** & **Flask** (Backend)
- **HTML & CSS** (Frontend)
- **Regex** for pattern-based malicious URL detection
- **Logging module** for event tracking
